Integrates H5P with WPLMS.

== Description ==

Use H5P components in WPLMS courses, quizzes and assignments.

1. Works with WPLMS Course ajax mode
2. Works with WPLMS Units and Quizzes

* Requires H5P for WordPress plugin :  <a href="https://wordpress.org/plugins/h5p/">https://wordpress.org/plugins/h5p/</a>

Refer Documentation : <a href="http://vibethemes.com/documentation/wplms/knowledge-base/wplms-h5p-addon/">http://vibethemes.com/documentation/wplms/knowledge-base/wplms-h5p-addon/</a>

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'WPLMS H5P'
3. Activate WPLMS H5P from your Plugins page. 

= From WordPress.org =

1. Download WPLMS H5P.
2. Upload the 'wplms-h5p' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate WPLMS H5P from your Plugins page.
